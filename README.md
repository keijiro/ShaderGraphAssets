# ShaderGraphAssets

Basic asset collection for Unity Shader Graph

## System requirements

- Unity 2022.3 or later
- Shader Graph 14 or later

## How to Install

You can install the ShaderGraphAssets package (`jp.keijiro.shadergraphassets`)
via the "Keijiro" scoped registry using the Unity Package Manager. To add the
registry to your project, follow [these instructions].

[these instructions]:
  https://gist.github.com/keijiro/f8c7e8ff29bfe63d86b888901b82644c
